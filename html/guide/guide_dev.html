마크업 안내

  • 아래 ID 빼고 대부분의 ID는 임시 ID 입니다. 변경해서 사용하셔도 됩니다. (예: popup, input, select 등등)
  • 변경하면 안되는 ID = 레이아웃 관련된 ID (#wrap, #header, #footer, #container, #contents 등등)
  • input, select, textarea의 id를 변경할때 label의 for 값도 같이 변경해두셔야합니다.(웹접근성이슈)
  • popup의 마크업은 꼭 #container 밖에 있어야하며, #header 나 #footer 안에 들어가서는 안됩니다.(웹접근성이슈)

버튼 비활성화

  • button클래스를 갖고 있으면 a태그를 button태그으로 바꾸거나, 반대로 button태그를 a태그로 바꿔도 무방합니다.

<a> 태그 일 경우

활성화 비활성화 활성화 비활성화
  • <a>에 "disabled" 클래스를 추가해주세요.
  • "disabled"클래스가 이벤트를 막아주지 않음. 별도의 if문 처리를 해줘야함

<button> 태그 일 경우

  • disabled 속성이 활성화되면 스타일이 변경됩니다.
  • disabled 속성은 고유속성으로 이벤트 실행이 안됩니다.(별도의 막는 스크립트 없음)

팝업

레이어팝업

종류

풀팝업 열기 모달팝업 열기 커튼팝업 열기

  • 팝업 ID 기준으로 호출하기때문에 팝업끼리 ID가 중복되면 안됩니다. 중복일 경우 변경해주시면 됩니다.
  • 팝업의 ID는 임시ID 입니다. 편한대로 변경해도 됩니다.
  • 팝업의 형태가 다르다 하여도 클래스로 구분되기때문에 모두 아래와 같은 방법으로 사용하시면됩니다.

오픈 방법

함수로 오픈
Layer.open('해당팝업ID');
  • 띄울려는 팝업의 ID를 넣어주면 됩니다.
클래스로 오픈
<a href="#해당팝업ID" class="button line ui-pop-open">모달팝업</a>
  • 클래스로 ui-pop-open를 넣어주시고 href속성에 팝업의 ID를 넣어주면 됩니다.
  • ui-pop-open클래스는 해당 용도가 아니면 에러를 발생할수 있으니 안쓸때는 삭제해주세요.

닫기 방법

함수로 닫기
Layer.close('해당팝업ID');
  • 숨기려는 팝업의 ID를 넣어주면 됩니다.
클래스로 닫기
<a href="해당팝업ID" class="button line ui-pop-close">팝업닫기</a>
또는 (닫으려는 팝업안에 있는 버튼이라면 클래스만 붙여도 작동)
<a href="#" class="button line ui-pop-close">팝업닫기</a>
  • 클래스로 ui-pop-close를 넣어주시고 href속성에 팝업의 ID를 넣어주면 됩니다.
  • 만약 닫으려는 팝업 안에 있는 버튼이라면 href속성에 팝업의 ID를 넣지 않아도 됩니다.
  • ui-pop-close클래스는 해당 용도가 아니면 에러를 발생할수 있으니 안쓸때는 삭제해주세요.

커스텀(옵션)

커스텀 얼럿창 팝업 열기 커스텀 컨펌창 팝업 열기
//커스텀 얼럿
Layer.alert({
    title:'얼럿창',
    content:'얼럿메세지<br>얼럿메세지',
    actionTxt:'닫기',
    action:function(){
        console.log('커스텀 얼럿 action');
    },
    callback:function(e){
        console.log('커스텀 얼럿 확인');
    }
});

//커스텀 컨펌
Layer.confirm({
    title:'<strong class="fc_blue">컨펌창</strong>',
    content:'컨펌메세지<br>컨펌메세지',
    actionTxt:'다음',
    action:function(){
        console.log('커스텀 컨펌 action');
    },
    cancelTxt:'이전',
    cancel:function(){
        console.log('커스텀 컨펌 cancle');
    },
    callback:function(e){
        if(e){
            console.log('커스텀 컨펌 확인');
        }else{
            console.log('커스텀 컨펌 취소');
        }
    }
});
